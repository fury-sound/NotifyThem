Простой интернет-пейджер на основе Firebase БД для отправки сообщений группам получателей. Примерный сценарий использования - преподаватель отправляет сообщения с домашним заданием ученикам по группам, ученики получают сообщения (только для своей группы). 
Ограничения: только текстовые сообщения.
Стек: SwiftUI, MVVM, Combine, async/await, Firebase
